modERec.Shower Class¶

class modERec.Shower¶

Bases: object

A class for the shower parameters.

ev¶

Unitary along \(\vec{v}\).

eB¶

Unitary along \(\vec{B}\).

evB¶

Unitary along \(\vec{v}\times\vec{B}\).

evvB¶

Unitary along \(\vec{v}\times\vec{v}\times\vec{B}\).

thetaCR¶

The shower zenith in deg.

phitCR¶

The shower azimuth in deg.

ECR¶

The cosmic ray energy

r_Core_proj¶

The position of the core projected into the shower plane.

bool_plot¶

Toggles the plots on and off.

Type

bool

d_Xmax¶

Distance to Xmax from the simultaion.
